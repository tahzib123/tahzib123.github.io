2020
  • React
  • Gatsby

Demo

Site that helps you find gifts for special occasions.
The data for the site is sourced from Contentful CMS and queried using GraphQL. Using Contentful's simple web interface anyone can add new products to the site.

Many performance considerations have been made such as prefetching all required data and lazy-loading it into the DOM. The site is also optimized for SEO and works as a Progressive Web App so it can function offline.
